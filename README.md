## Overview

binnincore is a lightweight C++ module that can be used as a foundation to build your Roblox executor.  
It comes with all necessities that you need for a Roblox internal. It includes the taskscheduler, offsets, luau and the start of the environment.

---

## Features

- Easy to build on  
- Lightweight and efficient.  
- Stable execution handling  
- Clean and modern structure  
- Ideal foundation for any custom executor  
- Easy to navigate  
- Blazing fast speeds

---

## Getting Started

### Software
- C++17 or newer  
- Visual Studio 2022 or Visual Studio Insiders  
- Understanding of Roblox internals  
- An Injector or timestamper if you want to use the WebView2Loader Injection method.
